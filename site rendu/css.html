LE CSS C'EST QUOI?

Cascading Style Sheets

C'est un langage informatique qui vient compléter un document HTML et permet de définir les styles associés à ces même documents. Il permet la création de site web responsive mais aussi de gérer la mise en forme et la mise ne page d'un site en choisissant la couleur du texte, la police utilisée, la taille du texte, les bordures, le fond… Il permet egalement, par exemple, d'indiquer où vous voulez que votre menu soit, sa largeur, que l'en-tête du site soit calé en haut et qu'il soit toujours visible, etc.

Histoire du CSS

Aux débuts du Web, CSS n'existait pas. En fait, il n'y avait initialement que le langage HTML. Le HTML est né en 1991 et CSS en 1996. La spécification CSS 1 final est donc publié en 1996 mais permet seulement de faire une mise en style, en modifiant la police et sa taille, la couleurs, les caractères, .. par exemple . CSS 2 est sortie en 1998 et a ajouté de nouvelles propriétés mais une version revue est sorti quelques temps après suite à des problèmes d’implémentation. Le CSS 3 est venue enrichir le CSS existant en englobant tous les aspects du CSS 1 et 2. Le CSS 3 est en évolution consente et de nouveaux modules sont régulièrement ajouté à la norme.

Exemples simple

Permet de mettre une couleur de fond et mettre en gras les paragraphes:

p {
background-color: #969696;
font-weight: bold;
}

Permet de changer la taille du titre et de le centrer

h1 {
font-size: 25px;
text-align: center;
}

Justifier le paragraphe dans la balise div et met une marge intérieur:

div p{
text-align: justify;
padding: 10px;
}
